To create a `package.json` file, we must use command `npm init` and confirm the different questions prompted.

```json
{
  "name": "package.json",
  "version": "1.0.0",
  "description": "descripcion",
  "main": "demo.js",
  "scripts": {
    "test": "echo \"Error: no test specified\" && exit 1"
  },
  "author": "lucasviera",
  "license": "ISC"
}
```
